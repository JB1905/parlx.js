Custom speed

To set custom speed to single element use following code in div tag:

<div data-speed="0.1">

To set custom speed to all elements place in JavaScript (or jQuery) options:

speed: 0.1

data-speed="0.3"

data-speed="-0.3"

Custom height

To set custom height to single element use following code in div tag:

<div data-height="700px">

To set custom height to all elements place in JavaScript (or jQuery) options:

height: '700px'

Disable parallax effect on mobile

To disable parallax effect on mobile on single element use following code in div tag:

<div data-mobile="false">

To disable parallax effect on all elements place in JavaScript (or jQuery) options:

mobile: false

data-mobile="false"

Parallax effect direction (horizontal, vertical or diagonal)

To disable parallax effect on mobile on single element use following code in div tag:

data-direction="vertical"

To disable parallax effect on all elements place in JavaScript (or jQuery) options:

direction: 'vertical'

data-direction="vertical"

data-direction="horizontal"

data-direction="diagonal"

Parallax effect type (foreground - move full div, background - move div content)

To set type of parallax effect use following code in div tag:

<div data-type="foreground">

To set type of parallax effect on all elements place in JavaScript (or jQuery) options:

type: 'foreground'

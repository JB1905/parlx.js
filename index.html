parlx.js

Parallax effect library for use with:
vanillaJS, CommonJS, jQuery and ES6

See on GitHub

Usage

At the beginning connect the library with Your project

• using script tag in HTML:

<script src="/directory/to/library/folder/parlx.js"></script>

• or via command line and CommonJS/ES6 import:

npm install parlx.js // npm

yarn add parlx.js // yarn
const Parlx = require('parlx.js'); // CommonJS

import Parlx from 'parlx.js'; // ES6

Next use library with

• Vanilla JavaScript e.g:

const elems = document.querySelectorAll('.parallax');
const parlx = new Parlx(elems, {
  // options...
});

• or jQuery:

Connect jQuery in HTML 
<script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>

or include via command line and CommonJS 
npm install jquery // npm

yarn add jquery // yarn

bower install jquery // bower
const jQuery = require('jquery');

and call plugin on element 
$('.parallax').parlx({
  // options...
});

• Plugin supports autoinit

To use it, add data-parlx to html element e.g:

<div data-parlx></div>

Custom speed

To set custom speed to single element use following code in div tag:

data-speed="0.1"

To set custom speed to all elements place in JavaScript (or jQuery) options:

speed: 0.1

data-speed="0.3"

data-speed="0"

data-speed="-0.3"

Custom height

To set custom height to single element use following code in div tag:

data-height="700px"

To set custom height to all elements place in JavaScript (or jQuery) options:

height: '700px'

Disable parallax effect on mobile

To disable parallax effect on mobile on single element use following code in div tag:

data-mobile="false"

To disable parallax effect on all elements place in JavaScript (or jQuery) options:

mobile: false

data-mobile="false"

Set parallax effect direction (horizontal, vertical or diagonal)

To disable parallax effect on mobile on single element use following code in div tag:

data-direction="vertical"

To disable parallax effect on all elements place in JavaScript (or jQuery) options:

direction: 'vertical'

data-direction="vertical"

data-direction="horizontal"

data-direction="diagonal"

Set parallax effect to background image or div

To set type of parallax effect use following code in div tag:

data-type="background"

To set type of parallax effect on all elements place in JavaScript (or jQuery) options:

type: 'background'
